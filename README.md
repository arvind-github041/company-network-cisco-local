# Alex Company Network Design (Cisco Packet Tracer)

## 📌 Project Description
This project is a complete enterprise network design created using Cisco Packet Tracer.  
The network is designed for alex.com company with multiple floors and servers.

## 🎯 Objective
- To design a real-world company network
- To configure different servers like DHCP, DNS, FTP, and Web Server
- To understand network communication and services

## 🏢 Network Overview
- 4 Floor network design
- Multiple PCs and switches
- Centralized servers

## 🌐 Servers Configured
- DHCP Server (Auto IP assignment)
- DNS Server (Domain resolution)
- FTP Server (File sharing with permissions)
- Web Server (Hosting website)

## 🔑 Key Features
- Automatic IP using DHCP
- FTP user-based permission system
- DNS record configuration
- Multi-floor structured network
